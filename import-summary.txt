Moved Files:
------------
Android Gradle projects use a different directory structure than ADT
Eclipse projects. Here's how the projects were restructured:

* AndroidManifest.xml => app/src/main/AndroidManifest.xml
* res/ => app/src/main/res/
* src/ => app/src/main/java/
* src/com/mediatek/camera/addition/remotecamera/service/ICameraClientCallback.aidl => app/src/main/aidl/com/mediatek/camera/addition/remotecamera/service/ICameraClientCallback.aidl
* src/com/mediatek/camera/addition/remotecamera/service/IMtkCameraService.aidl => app/src/main/aidl/com/mediatek/camera/addition/remotecamera/service/IMtkCameraService.aidl
* src/com/mediatek/camera/v2/detection/README => app/src/main/resources/com/mediatek/camera/v2/detection/README
* src/com/mediatek/camera/v2/detection/facedetection/README => app/src/main/resources/com/mediatek/camera/v2/detection/facedetection/README
